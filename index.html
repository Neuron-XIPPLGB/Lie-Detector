Proyek: Arduino Lie Detector

Mengungkap rahasia di balik denyut nadi dan respons kulit.

Jelajahi Proyek

Detail Proyek

Proyek ini adalah demonstrasi sederhana alat detektor kebohongan (poligraf) yang dibuat menggunakan platform Arduino. Alat ini mengukur dua parameter fisiologis utama yang sering dikaitkan dengan respons stres: **Galvanic Skin Response (GSR)** dan **detak jantung**.

Komponen yang Digunakan:

  • Arduino Board
  • Sensor GSR
  • Sensor Detak Jantung
  • Breadboard & Kabel Jumper

Skema rangkaian proyek.

// Contoh Sketsa Arduino
void setup() {
  Serial.begin(9600);
}

void loop() {
  int gsrValue = analogRead(A0);
  Serial.print("GSR:");
  Serial.print(gsrValue);
  Serial.println();
  delay(500);
}
